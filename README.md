# GloryGames

GloryGames is a mobile-first browser gaming platform built for instant play, high scores, daily challenges, game categories, leaderboards, and friendly competition.

The platform is designed to host HTML5 browser games that work across desktop, tablet, and mobile devices.

## Features

- Free browser-based games
- Mobile-first gameplay experience
- Game categories such as Arcade, Action, Adventure, Educational, Puzzle, Racing, Casual, and more
- Game detail pages with SEO metadata
- Play pages with embedded games
- Daily challenge system
- Leaderboards and score tracking
- Game chat using realtime sockets
- SEO support with sitemap and robots files
- Legal and platform pages including Privacy Policy, Terms of Use, Cookie Policy, Community Guidelines, Parents & Safety, Accessibility, About, Contact, Submit a Game, and Advertise

## Tech Stack

- Next.js App Router
- React
- TypeScript
- Prisma ORM
- PostgreSQL
- Redis
- Socket.IO realtime server
- HTML5 games served from `public/games`
